# Session Pack - 세션 인계서 생성 및 정리

현재 세션의 상태를 정리해서 다음 세션에 넘겨줄 인계서를 만들어주세요.

## 작업 내용
1. **CLAUDE.md 업데이트**: 이번 세션에서 변경된 프로젝트 규칙이나 구조가 있으면 반영
2. **현재 작업 상태 기록**: `.claude/handoff.md`에 다음 내용을 기록
   - 현재 진행 중인 작업과 완료율
   - 이번 세션에서 내린 주요 결정들
   - 다음 세션에서 이어갈 작업 (Next Steps)
   - 알려진 이슈나 주의사항
3. **배운 것 정리**: 이번 세션에서 발견한 패턴이나 주의할 점이 있으면 기억에 저장

## 출력 형식
간결하게, 다음 세션의 Claude가 읽고 바로 이어갈 수 있는 형태로 작성하세요.

## 완료 후 안내
모든 파일 저장이 끝나면 다음과 같이 안내하세요:

✅ Pack 완료!
- 이어서 작업하려면 → `/compact`
- 오늘 끝이면 → `/clear`
